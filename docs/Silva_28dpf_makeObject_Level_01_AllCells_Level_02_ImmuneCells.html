Introduction

The purpose of this analysis is to analyse a dataset that is specifically enriched for mpeg1.1+ cells in the zebrafish brain. Our lab is mostly interested in endothlial cells, so we tend to sort for them. Getting publicly available data from this paper where they sequence 2 juvenile and 1 adult zebrafish immune cells in the brain (CD45+ mpeg1.1+ FACS).

I have download their filtered data they deposited on GEO (GSE164772).

make_density_plots <- function(seurat =Level_01_seuratObject, sample){
  plot_list <- list()

  #ncount
  p1 <- ggplot(seurat@meta.data %>% filter(., individual == sample), aes(x=log10(nCount_RNA))) + 
    geom_histogram(aes(y=..density..), colour="black", fill="white")+
    geom_density(alpha=.2, fill="#FF6666")+ xlab('log10(count)') +
    plot_sizing + ggtitle(sprintf("Library size, %s", sample))
  
  #nfeat
  p2 <- ggplot(seurat@meta.data %>% filter(., individual == sample), aes(x=log10(nFeature_RNA))) + 
    geom_histogram(aes(y=..density..), colour="black", fill="white")+
    geom_density(alpha=.2, fill="#FF6666")+ xlab('log10(nfeature)') +
    plot_sizing + ggtitle(sprintf("#features, %s",  sample))
  
  #mito
  p3 <- ggplot(seurat@meta.data %>% filter(., individual == sample), aes(x=percent.mt)) + 
    geom_histogram(aes(y=..density..), colour="black", fill="white")+
    geom_density(alpha=.2, fill="#FF6666")+ xlab('percent.mt') +
    plot_sizing + ggtitle(sprintf("mito, %s",  sample))
  
  return(p1+p2+p3)
  
}

make_correlation_plots <- function(seurat =Level_01_seuratObject, sample){
  plot_list <- list()

  meta <- seurat@meta.data %>% 
    filter(.,  individual == sample) %>% 
    mutate(., log10ncount = log10(nCount_RNA),
              log10nfeat = log10(nFeature_RNA))
  
  #ncount
  p1 <- ggscatter(meta, x = "log10ncount", y = "log10nfeat",
          add = "reg.line",                                 
          conf.int = TRUE,                                
          add.params = list(color = "red4",
                            fill = "lightgray")) +
    plot_sizing + ylab("log10(#features)") + xlab("log10(nCount)") +
    stat_cor(method = "pearson", label.x = 3, label.y = 4, size = 5) +
    ggtitle(sprintf("Lib size vs #features, %s", sample))
  
  #nfeat
  p2 <- ggscatter(meta, x = "log10ncount", y = "percent.mt",
          add = "reg.line",                                 
          conf.int = TRUE,                                
          add.params = list(color = "red4",
                            fill = "lightgray")) +
    plot_sizing + ylab("%mito") + xlab("log10(nCount)") +
    stat_cor(method = "pearson", label.x = 3, label.y = 80, size = 5) +
    ggtitle(sprintf("Lib size vs mito, %s",  sample))
  
  #mito
  p3 <- ggscatter(meta, x = "log10nfeat", y = "percent.mt",
          add = "reg.line",                                 
          conf.int = TRUE,                                
          add.params = list(color = "red4",
                            fill = "lightgray")) +
    plot_sizing + ylab("%mito") + xlab("log10(#features)") +
    stat_cor(method = "pearson", label.x = 2.6, label.y = 80, size = 5) +
    ggtitle(sprintf("#features vs mito, %s",  sample))
  
  return(p1+p2+p3)
  
}

Compute region specific signatures scores - putative CAM/MG

So the paper where this data is from, they claim to discover spatially different microglia they call NAM and SAM. As we’re using their clusters, I can directly link them here..

JM1 (maps to one of our MG cluster) -> NAMs (Optic tectum genes)

JM4 (maps to our CAM cluster) -> SAMs (Hind brain genes)

They use their own bulk RNA-seq to derive region specific signatures, I’ve downloaded the data and replicated it.

The filters they used were:

adj. p < 0.05, log2 fold change >1.2, basemean >100
